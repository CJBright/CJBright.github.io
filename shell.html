Study Notes.

File System.

Paths can be specified as both absolute and relative. Absolute paths start with /.
/home/user/file.

Relative paths go from the current directory.


Hover to see the 7 columns from the ls -l command.
1. Permissions.
2. Number of links.
3. Owner.
4. Group.
5. Size.
6. Date.
7. Name.


The file type can be determined from the first letter of the permissions:
- = file.
l = link.
d = directory.
c = special/device file.
s = socket.
p = named pipe.
b = block device.

Command: Action:
cd cd Directory -> Change into relative path directory.

cd /path -> Change to directory at absolute path.

cd .. -> Go into parent directory.
Change directory (~).
ls ls /path -> List items in the given path.

ls -l -> Print items with more information, permission, time etc.

ls -tr -> Organises by time created, then reverses (oldest-newest).

ls -i -> Adds inode information.
List items in current directory.
pwd Print working directory.
touch touch file -> Creates file.

touch file1 file2 file3 -> Creates multiple files.

touch file{1..10} -> Creates 10 files.
Creates an empty file.
cp (-R) cp file2copy copiedName -> Creates a copy of the file to a new filename.

cp dir2copy copiedName -> Creates a copy of the directory to a new directory name.

Copies a file (or Directory).
mv mv file-oldname file-newname -> Renames the file.

mv /old/path/file /new/path/file -> Reocates the file.

Move or rename a file/directory.
vi vi filename -> Opens Vi and creates a file.

Quit by pressing Esc, then typing :wq!, enter.
Move or rename a file/directory.
mkdir mkdir dirname -> Creates a new directory.
Creates an empty directory.
rmdir (-r) rmdir dirname -> Removes the specified directory.
Removes an empty directory (not empty).
